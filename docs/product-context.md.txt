## Что строим

### Продукт 1: Аудит безопасности «Безопасный бизнес»

Сервис для владельцев малого бизнеса (старт — студии маникюра).
Помогает защититься от штрафов при проверках Роспотребнадзора, пожарной, трудовой инспекции.

**Как работает:**

- Бесплатный тест (12 вопросов) → результат на странице с индексом безопасности
- Платный полный аудит → персональная страница с картой нарушений, дорожной картой устранения
- PDF-отчёт можно скачать дополнительно
- Монетизация: платный аудит внутри платформы

**Готово:**

- 15 файлов ТЗ (архитектура, сегментация, вопросы, логика баллов, матрица 29 нарушений, карта уязвимостей, дорожная карта, обязательные документы, персональные рекомендации)
- Пример PDF-отчёта (13 страниц)
- Демо-версия PDF с размытыми данными

-----

### Продукт 2: Studio OS — операционная система для студий красоты

**Концепция:** Всё что нужно владельцу студии в одном месте. Аудит безопасности встроен как раздел.

**Стек:** React (frontend) + Node.js + Express (backend) + PostgreSQL

**Роли:**

- Super Admin (Артём) — видит все студии всех клиентов платформы
- Владелец студии — все свои данные, несколько студий в одном аккаунте
- Мастер — только свои визиты, клиенты без телефона, своя зарплата

**Две студии Артёма:**

- Студия на Тверской
- Студия на Арбате
- У Артёма и супруги — права владельца в обеих студиях через свои аккаунты

**Разделы платформы:**

### ЛК Владельца (нижняя навигация):

Главная / Клиенты / Визиты / Финансы / Ещё

**Ещё:** Склад расходников / Чек-листы смены / База знаний / Безопасность / Обратная связь (от мастеров) / Команда / Настройки

### ЛК Мастера (нижняя навигация):

Главная / Клиенты / Смена / Склад / Финансы / Ещё

**Ещё:** База знаний / Обратная связь владельцу

## Ключевые требования к платформе

### Форма визита (критично важно):

- Фото до/после раздельно
- Фамилия (Enter→имя), Имя (Enter→услуга) — быстрый переход
- Услуга
- Материалы
- Сумма без скидки
- Скидка в % (рядом автоматический пересчёт в ₽)
- Дата и время (автоматически, редактируемые)
- Заработок мастера = 40% от ПОЛНОЙ суммы без учёта скидки
- Процент выплаты индивидуален для каждого мастера
- При вводе фамилии — поиск клиента (без показа телефона мастеру!)
- Если клиент найден — предложить “Это он/она?” или “Другой человек”

### Безопасность данных:

- Телефон клиента мастер НЕ видит НИКОГДА — на уровне API
- Данные аудита безопасности хранятся отдельно и зашифрованы
- Изоляция данных между студиями (студия А не видит данные студии Б)

### Финансы владельца:

- Выручка по периодам (сегодня/неделя/месяц)
- Зарплаты мастеров (авторасчёт, % от выручки каждого, от ПОЛНОЙ суммы)
- Постоянные расходы (фикс ₽): аренда, DIKIDI, Studio OS
- Процентные расходы (% от выручки): налог УСН 6%, эквайринг 1.5%
- Переменные расходы (фикс ₽): расходники, кухня
- Чистая прибыль = Выручка − Зарплаты − Пост.расходы − (Выручка × % расходы) − Перем.расходы

### Чек-листы:

- Редактирует ТОЛЬКО владелец
- Мастер только отмечает выполнение

### База знаний:

- Добавлять разделы и статьи может ТОЛЬКО владелец
- Мастер только читает

### Команда:

- Приглашение мастеров по ссылке
- Удалить мастера может только владелец (с подтверждением)

### Обратная связь:

- Мастер пишет → владелец видит в разделе “Обратная связь”

### Расходники:

- Ссылка на товар к каждой позиции
- Кнопки “Пришло” и “Списать”
- Индикатор низкого остатка

### Безопасность (раздел в платформе):

- Встроен полный аудит “Безопасный бизнес”
- Бесплатный тест 12 вопросов → результат на странице
- Платный аудит → персональная страница (не PDF)
- Загрузка PDF-отчёта
- Карта нарушений с возможностью отметить “Устранено”
- Хранение документов студии по категориям (из файла 13_Mandatory_Documents)
