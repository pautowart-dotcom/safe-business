СТРУКТУРА ШАБЛОНОВ ЖУРНАЛОВ — "Безопасный бизнес"

Каждая папка = один тип журнала:
  uf-lamp/              — Журнал УФ-бактерицидной установки (A5, альбомная)
  sterilizers/           — Журнал контроля работы стерилизаторов (A4, альбомная)
  pre-sterilization/     — Журнал предстерилизационной обработки (A4, альбомная)
  instruktazh/            — Журнал регистрации инструктажа (A4, альбомная)
  disinfectants/          — Книга учёта дезсредств (A4, альбомная)

В каждой папке:
  preview.pdf   — готовый пример для просмотра (что видит человек)
  source/       — HTML + CSS файлы, из которых генерируется PDF.
                  base.css — общий стиль (цвета, шрифты, структура таблиц).
                  cover.html — обложка (сюда встраивается QR при генерации).
                  остальные .html — внутренние страницы.
                  assets/ — логотип.

Для новых типов журналов, которых ещё нет — см. папку _placeholder-example/
