Пример заглушки для журнала, дизайн которого ещё не готов.

Когда появится новый тип журнала без готового дизайна — создать такую же папку
по имени типа (например: uborka, medknizhki и т.д.), положить в неё этот файл,
и указать в списке типов журналов в приложении с пометкой "скоро будет".

Как только дизайн будет готов — просто заменить содержимое папки на настоящие
source/ (HTML+CSS) и preview.pdf по аналогии с остальными журналами рядом.
